Some facts y beneficios de Flex

  • Flexbox nació en 2009 para proveer un nuevo sistema de layouts conocido como "flex layout"
  • Flexbox o Flex es un ecosistema completo, distinto a otras múltiples propiedades de CSS3 que son independientes
  • Flex nos permite organizar el flujo de los elementos de nuevas maneras
  • Por fin podemos alinear elementos verticalmente de manera simple

Macro Flex (aplicado al contenedor)

Macro Flex

Propiedad: display

Es la propiedad más importante en el ecosistema the Flex layouts.

Sin esta, todas las otras no van a funcionar.

.wrapper { display: flex || inline-flex; }

Ver más sobre display

Macro Flex

Propiedad: flex-direction

Se encarga de orientar los elementos contenidos.

.wrapper { flex-direction: row || row-reverse || column || column-reverse; }

Ver más sobre flex-direction

Macro Flex

Propiedad: flex-wrap

Dependiendo de esta propiedad los elementos se manejan en una sola línea o en múltiples. Bastante similar al efecto de white-space en elementos inline.

.wrapper { flex-wrap: nowrap || wrap || wrap-reverse; }

Ver más sobre flex-wrap

Macro Flex

Propiedad: flex-flow

Shorthand que combina flex-direction con flex-wrap.

.wrapper { flex-flow: {flex-direction} {flex-wrap}; }

Ver más sobre flex-flow

Macro Flex

Propiedad: justify-content

Controla horizontalmente la posición de los elementos, si la dirección es tipo row. De ser column, los posiciona verticalmente.

.wrapper { justify-content: flex-start || flex-end || center || space-around || space-between; }

Ver más sobre justify-content

Macro Flex

Propiedad: align-items

Similar a justify-content, controla verticalmente la posición de los elementos, si la dirección es tipo row. De ser column, los posiciona horizontalmente.

Su efecto solo tiene lugar cuando los elementos están en la misma línea.

.wrapper { align-items: flex-start || flex-end || center || stretch || baseline; }

Ver más sobre align-items

Macro Flex

Propiedad: align-content

Similar a justify-content y a align-items, controla verticalmente la posición de los las filas de elementos, si la dirección es tipo row.

No tienen ningún efecto si solamente hay una fila.

.wrapper { align-content: flex-start || flex-end || center || stretch || space-around || space-between; }

Ver más sobre align-content

Micro Flex (aplicado a los elementos)

Micro Flex

Propiedad: flex-grow

Proporción de crecimiento con respecto a los elementos hermanos. La medida se configura en números naturales y el cero (valor por defecto, el elemento no crece).

.wrapper .elements { flex-grow: 1; /* Default 0 */ } .wrapper .elements:nth-child(5) { flex-grow: 2; }

Ver más sobre flex-grow

Micro Flex

Propiedad: flex-shrink

Similar a flex-grow, esta propiedad establece la relación de decrecimiento con respecto a sus elementos hermanos. La medida se configura en números naturales y el cero (valor por defecto, el elemento no decrece).

.wrapper .elements { flex-shrink: 1; /* Default 0 */ } .wrapper .elements:first-child { flex-shrink: 2; }

Ver más sobre flex-shrink

Micro Flex

Propiedad: flex-basis

Tamaño fijo o proporcional que se le asigna a un elemento.

Similar a width, asigna un tamaño previo a distribuir el espacio disponible del contenedor.

Hint: flex-basis no soporta calc() ni box-sizing en Internet Explorer 10 ni 11. Si necesitan soporte es preferible no definir esta propiedad y usar width.

.wrapper .element { flex-basis: 100px; /* Default auto */ }

Ver más sobre flex-basis

Micro Flex

Propiedad: flex

Shorthand que combina flex-grow, flex-shrink y flex-basis.

Hint: Si van a usarflex asegúrense de definir los tres valores (aunque sean sus valores por defecto). Las versiones más viejas de los navegadores pueden romper el layout por inconvenientes de compatibilidad.

.wrapper .element { flex: {flex-grow} {flex-shrink} {flex-basis}; }

Ver más sobre flex

Micro Flex

Propiedad: align-self

Podemos decir que es el equivalente a align-items pero aplicado directamente al elemento.

Controla verticalmente la posición del mismo, si la dirección es tipo row. De ser column, lo posiciona horizontalmente.

Su efecto solo se aplica en el espacio de su propia fila, sin interferir con otras.

.wrapper .element { align-self: flex-start || flex-end || center || stretch || baseline; }

Ver más sobre align-self

Micro Flex

Propiedad: order

Nos permite brindar un orden a los elementos sin tener que modificar el DOM. Los valores posibles son números enteros, siendo cero el valor por defecto.

.wrapper .elements { order: 1; /* Estos elementos se posicionan al final */ } .wrapper .elements:last-child { order: -1; /* Este elemento se posiciona al comienzo */ }

Ver más sobre order

Algunas consideraciones

La sintaxis de Flexbox cambió muchas veces desde el primer spec y para maximizar la compatibilidad con navegadores más antiguos como Internet Explorer 10 y 11 hacen falta algunas consideraciones.

Algunas consideraciones

Sintaxis año 2009

.wrapper { display: box; /* Y sus variantes -webkit-, -moz-, -ms-, -o- */ }

Sintaxis año 2011

.wrapper { display: flexbox; /* Y sus variantes -webkit-, -moz-, -ms-, -o- */ }

Sintaxis actual

.wrapper { display: flex; /* Y sus variantes -webkit-, -moz-, -ms-, -o- */ }

Ver más sobre sintaxis
